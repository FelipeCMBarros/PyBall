Nome do Jogo - PyBall: Jogo de Pênaltis

Integrantes do grupo - Felipe Barros, Mateus Fischel e Paulo Mello

Descrição - É um jogo de disputa de pênaltis desenvolvido em Python com Pygame, onde você enfrenta um goleiro em uma emocionante competição de 5 rodadas! O jogo simula cobranças de pênaltis realistas com 6 zonas clicáveis no gol (3 superiores e 3 inferiores). A cada rodada, você escolhe onde chutar clicando em uma zona, enquanto o goleiro tenta adivinhar sua escolha. O jogo tem sliders interativos, um slider de força que oscila automaticamente (segure e solte no timing certo) e um slider de direção que se move horizontalmente para definir o desvio lateral da bola. Quanto mais forte o chute, mais rápido o slider de direção! Vence quem marcar mais gols nas 5 rodadas.

Dependências - Esse jogo é dependente da biblioteca pygame do python. Para instalar, entre no link: https://www.pygame.org/download.shtml

Para rodar o jogo basta rodar o script 'python main.py' com Python e Pygame instalados.
